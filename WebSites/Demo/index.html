Embedded

This site will demonstrate different ways to integrate the generated video into content online

First step is to get the movie file online so that it can be referred by other sites. Most websites can host the video directly, as an alternative it can be copied storage services such as dropbox

To learn more about the different video tags and browser support go to this site

To show the video directly into your own site, copy the following code snippet

<video src="{path to video}" type="video/mp4" autoplay loop></video>

Mouse hover

To run a version that only plays when the mouse hovers the video put in the following code snippet

The javascript will pick up all div elements with the id called playOnMouseHover and play video inside them on mouse hover and pause on mouse leave

<div class="playOnMouseHover center" style="display: inline-block">
   <video src="../test.mp4" type="video/mp4" loop></video>	
</div>
...
<script src="js/videoUtils.js"></script>

Play in view

To run a version that will start playing when the video is fully visible in the browser add the following code snippet

The javascript will pick up all video elements with the id called playOnEnterView and play the video when it is fully visible in the browser

   <video src="../test.mp4" type="video/mp4" loop></video>	
...
